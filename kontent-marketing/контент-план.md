## АУДИТОРИЯ — СЕГМЕНТЫ И БМПХС

### Сегмент 1 — Владелец малого бизнеса (5–30 человек)
*Строитель, студия, сервис, небольшое производство, агентство*

| | |
|---|---|
| **Боли** | Всё держится на нём. Уходит — рассыпается. Сотрудники не принимают решений сами. |
| **Мечты** | Бизнес работает без него. Уехать в отпуск и не получать звонки. Зарабатывать больше при тех же людях. |
| **Проблемы** | Нет системы задач, нет контроля, деньги утекают непонятно куда. Наём не помогает — больше людей = больше хаоса. |
| **Хотелки** | Простое решение, не "трансформация бизнеса". Конкретный шаг, а не консалтинг на полгода. |
| **Страхи** | Потеря контроля. Бизнес без него умрёт. Ещё одно дорогое внедрение, которое не взлетит. |

### Сегмент 2 — Эксперт-фрилансер / соло-предприниматель
*Консультант, коуч, дизайнер, маркетолог — работает сам на себя*

| | |
|---|---|
| **Боли** | Продаёт своё время, а не результат. Потолок дохода — 24 часа в сутках. Всё в голове, ничего не зафиксировано. |
| **Мечты** | Выйти из продажи времени. Делегировать рутину. Сделать систему, которая работает без его постоянного присутствия. |
| **Проблемы** | Не умеет/боится нанимать. Не знает, что автоматизировать первым. Пробовал инструменты — не прижились. |
| **Хотелки** | AI-инструменты, которые реально экономят время. Готовые шаблоны и схемы, а не теория. |
| **Страхи** | Потерять клиентов если замедлится. Потратить деньги на инструменты и не получить результат. |

### Сегмент 3 — Стартапер / основатель на ранней стадии
*Команда 2–7 человек, продукт есть, но всё держится на основателе*

| | |
|---|---|
| **Боли** | Масштабироваться хочет, но процессов нет. Нанял первых людей — стало сложнее, а не легче. |
| **Мечты** | Выстроить основу до того, как масштаб сломает всё. Привлечь инвестиции с порядком внутри. |
| **Проблемы** | Некогда думать о системах — надо закрывать задачи. "Позже разберёмся с процессами" — и не разбираются. |
| **Хотелки** | Минимальная система для маленькой команды. Быстро, дёшево, без enterprise-решений. |
| **Страхи** | Нанять людей, которые работают плохо и не смочь уволить. Потерять управляемость при росте. |
