### Вывод по конкурентам

**Что есть на рынке:**
- Крупные корпоративные инструменты (IBM, Gartner) — только для Enterprise, за paywall, только English
- Маркетинговые опросники вендоров (Microsoft, Cisco, Sopra Steria) — бесплатно, но результат = реклама их продуктов, без реального роадмапа
- Отраслевые рейтинги (Evident) — только для гигантов, нет самооценки

**Незанятая ниша для K-A-Q-S™:**
- На **русском языке** — рынок полностью свободен
- **Независимая методология** — не продаёт чужой продукт, а даёт объективную картину
- **Конкретный роадмап** — не «ваш балл 67», а «сделайте X, Y, Z в ближайшие 90 дней»
- **PDF-отчёт** — ни один бесплатный конкурент не даёт скачиваемый документ
- **SMB-фокус** — компании 20–500 человек, которых игнорируют Gartner и IBM

## 3. UX и сценарии

### Сценарий 1: Руководитель («Хочу понять, где мы стоим»)
- **Персона:** CEO/COO компании 50–200 человек
- **Триггер:** Прочитал статью про ИИ, конкурент внедрил ИИ-агента
- **Путь:** Лендинг → Регистрация → Опросник (20 мин) → Результаты → PDF → Покупка консультации
- **Ожидание:** «Скажите мне что делать, а не что такое ИИ»
- **Боли:** Нет времени разбираться в технологиях, непонятна ROI

### Сценарий 2: Стратег («Нужна база для совета директоров»)
- **Персона:** CDO/CTO, выступает перед советом
- **Триггер:** Нужен авторитетный документ с оценкой
- **Путь:** Лендинг → Оценка → Скачивает PDF → Показывает совету
- **Ожидание:** Профессиональный отчёт с методологией
- **Боли:** Внутренние оценки не воспринимаются как объективные

### Сценарий 3: Консультант («Оцениваю клиентов»)
- **Персона:** AI-консультант, фрилансер или агентство
- **Триггер:** Нужен инструмент для диагностики клиентов
- **Путь:** Подписка → White-label → Использует в проектах
- **Ожидание:** Инструмент под своим брендом
- **Боли:** Нет готового инструмента, создавать с нуля дорого

### Топ-7 хороших UX-паттернов
1. **Прогресс-бар с шагами** — пользователь видит конец туннеля
2. **Автосохранение + «продолжить позже»** — снижает тревогу незавершённости
3. **Объяснение «почему мы спрашиваем»** — повышает качество ответов
4. **Мгновенный фидбэк после каждого модуля** — микро-результат удерживает внимание
5. **Конкретный следующий шаг после результатов** — не «ваш уровень 3», а «сделайте X»
6. **PDF по email** — повышает ценность, пользователь не теряет результат
7. **Social proof с числами** — «237 компаний прошли оценку»

### Топ-7 UX-ошибок
1. Слишком много вопросов без видимого прогресса
2. Требовать корпоративный email до показа результатов
3. Результаты без объяснения как исправить
4. Не сохранять черновик — потеря данных при закрытии
5. Мобильная версия как afterthought
6. PDF без логотипа и даты — выглядит как самодельный
7. Нет возможности исправить ответ назад

## 4. Технологии

### База данных
| Вариант | Плюсы | Минусы | Вывод |
|---------|-------|--------|-------|
| **PostgreSQL + Prisma** | Надёжен, SQL, Prisma ORM | Нужен хостинг | ✅ MVP |
| **Supabase** | PostgreSQL + auth + realtime из коробки | Vendor lock | ✅ Хорошо для старта |
| PlanetScale | Serverless MySQL | Закрыли бесплатный план | ❌ |
| MongoDB Atlas | Гибкая схема | JSON-ответы уже есть, SQL предпочтительнее | Не нужен |

**Рекомендация:** Supabase (PostgreSQL managed) на старте, Prisma как ORM.

### Хостинг
| Вариант | Free tier | Плюсы | Минусы |
|---------|-----------|-------|--------|
| **Vercel** | Есть | Next.js родной, CI/CD из коробки | Serverless ограничения |
| Railway | $5/мес | Docker, PostgreSQL рядом | Чуть сложнее |
| Render | Есть | Просто, PostgreSQL | Cold start на free |
| VPS (Hetzner/DO) | От €4/мес | Полный контроль | Нужно DevOps |

**Рекомендация:** Vercel (Next.js) + Supabase (DB) — самый быстрый старт.

### Аутентификация
| Вариант | Плюсы | Минусы |
|---------|-------|--------|
| **Custom JWT (jose + bcrypt)** | Полный контроль, уже написан | Нужно следить за безопасностью |
| NextAuth.js | OAuth провайдеры из коробки | Сложная конфигурация |
| Clerk | UI компоненты, email, MFA | $25/мес при росте, vendor lock |

**Рекомендация:** Custom JWT на MVP — уже реализован, достаточно.

## 5. Монетизация

### Модель 1: Freemium
- **Бесплатно:** 1 оценка, базовые результаты (без деталей)
- **Платно:** Полный отчёт, PDF, роадмап, история
- **Конверсия:** 2–5% free → paid
- **Плюс:** Вирусность, низкий барьер входа
- **Минус:** Нужно много трафика

### Модель 2: Подписка ⭐ Рекомендуется для старта
| Тариф | Цена/мес | Что включено |
|-------|----------|--------------|
| Starter | 2 900 ₽ | 1 оценка/мес, PDF, роадмап |
| Growth | 7 900 ₽ | 3 оценки/мес, бенчмарк, история |
| Pro | 19 900 ₽ | Безлимит, white-label, API |

- **Путь к $10k MRR:** 35 клиентов на Growth (7 900 ₽) или 17 на Pro (19 900 ₽)
- **Аналоги:** Typeform, Hotjar — обе выросли через подписку

### Модель 3: Pay-per-report
- **Цена:** 4 900–14 900 ₽ за отчёт
- **Плюс:** Нет обязательств для клиента
- **Минус:** Непредсказуемый доход

### Модель 4: White-label для консультантов
- **Цена:** 29 900 ₽/мес + брендинг
- **Клиент:** Консалтинговые агентства, фрилансеры
- **Потенциал:** 10 агентств × 30k = 300k ₽/мес

### Модель 5: Лидогенерация
- **Суть:** Бесплатная оценка → продажа консультации Карины
- **Конверсия:** 10–20% (высокий intent после оценки)
- **Цена консультации:** 50–150k ₽
- **Плюс:** Синергия с личным брендом

### Модель 6: Партнёрства
- **Суть:** Интеграция в пакеты IT-компаний, банков, акселераторов
- **Формат:** Revenue share 20–30% или фиксированная лицензия
- **Примеры:** Сбер, Т-Банк (для МСП), SkillFactory, Нетология

### Модель 7: Enterprise
- **Цена:** От 150 000 ₽/год
- **Что включает:** Кастомные вопросы, SLA, обучение, выгрузка данных
- **Цикл сделки:** 3–6 месяцев
- **Фокус:** После запуска и 50+ клиентов

### Рекомендуемая стратегия
```
Месяц 1–3:  Freemium → лидогенерация консультаций
Месяц 4–6:  Платные отчёты (pay-per-report)
Месяц 7–12: Подписка (Starter + Growth)
Год 2+:     White-label + Enterprise + Партнёрства
```

### Путь к первым $10k MRR
1. **Личный бренд → трафик:** TG-канал, LinkedIn, выступления
2. **Бесплатные оценки → конверсия в консультации:** 50k–150k ₽/клиент
3. **3–5 первых платящих клиентов → кейсы и социальное доказательство**
4. **Запуск подписки с ранним доступом:** скидка 50% первым 20 клиентам
5. **Реферальная программа:** консультанты рекомендуют инструмент

### Риски
- **Open Risk:** Крупный игрок (Сбер AI, Яндекс) может запустить похожий продукт
  - Ответ: Скорость + личный бренд + методология К-A-Q-S™ как торговая марка
- **Open Risk:** Низкая конверсия из бесплатного в платное
  - Ответ: Сначала монетизация через консультации, не SaaS
