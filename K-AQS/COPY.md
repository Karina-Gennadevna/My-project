## ЭКРАН 3 — K-A-Q-S Framework

### Заголовок

```
Четыре оси управляемости.
```

### Подзаголовок

```
Управляемый бизнес — это не контроль.
Это система, которая работает без ручного надзора
и масштабируется без роста хаоса.

K-A-Q-S™ измеряет четыре компонента этой системы.
```

### Четыре блока

**K — Knowledge Architecture**
```
Архитектура знаний

Задокументированы ли процессы.
Есть ли база знаний, независимая от людей.
Насколько бизнес зависит от «незаменимых».

Если ключевой человек уйдёт завтра —
что именно потеряет компания.
```

**A — Automation Readiness**
```
Готовность к автоматизации

Описаны ли процессы достаточно,
чтобы их автоматизировать без переделки.
Стандартизированы ли входы и выходы.

Автоматизация без этого — ускорение ошибки,
а не ускорение роста.
```

**Q — Quality & Risk Control**
```
Качество и управление рисками

Как выявляются отклонения от нормы.
Есть ли система раннего обнаружения рисков.
Как принимаются решения при неопределённости.

Качество — это не проверка после факта.
Это архитектура, которая предотвращает факт.
```

**S — Scalability**
```
Масштабируемость

Выдержит ли операционная модель рост ×2.
Где узкие места, которые не видны при текущем масштабе.
Что сломается первым при увеличении нагрузки.

Масштаб проверяет систему.
K-A-Q-S™ — проверяет раньше.
```

## Словарь: что не использовать

Эти слова и формулировки не появляются в текстах K-A-Q-S™:

| Не писать | Почему |
|-----------|--------|
| «Мощный инструмент» | Прилагательное без содержания |
| «Уникальная методология» | Любой так говорит |
| «Революционный подход» | Не относится к управлению бизнесом |
| «Помогаем бизнесам расти» | Это слоган агентства, не платформы |
| «Наши эксперты» | Нет экспертов, есть система |
| «Индивидуальный подход» | Если так говорят все — это не дифференциатор |
| «Оставьте заявку» | Не заявку — диагностику |
| «Мы поможем вам» | Платформа не «помогает» — она измеряет |
| «Уже доверяют X компаний» | Только с реальной цифрой, иначе — не писать |
| «Попробуйте бесплатно» | Не «попробуйте» — «пройдите диагностику» |
| «Трансформация бизнеса» | Корпоративный новояз |
| «Синергия» | Нет |
| «Экосистема» | Нет |
| «Дорожная карта» | Писать «роадмап» или «план на 90 дней» |
