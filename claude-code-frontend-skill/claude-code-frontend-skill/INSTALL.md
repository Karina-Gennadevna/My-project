# Скилл: Lovable-качество фронтенда с первого запроса

Скилл для **Claude Code** — превращает ИИ в команду из дизайнера, фронтенд-разработчика и копирайтера.
Результат: production-ready интерфейс с реальными фото, анимациями и настоящим текстом — с первого запроса.

---

## Что входит в пакет

| Файл | Что делает |
|------|-----------|
| `frontend-design/SKILL.md` | Главный скилл — правила дизайна, шрифты, цвета, компоненты, копирайтинг |
| `FRONTEND_DESIGN_SYSTEM.md` | Справочник: готовые font-теги, URL фотографий, CSS-паттерны для быстрого доступа |
| `DESIGN_SYSTEM_NEON.md` | Neon Cyberpunk тема — для тёмных проектов, дашбордов, Mini Apps |

---

## Требования

- **Claude Code** установлен и авторизован
  - Установить: https://claude.ai/download (или `npm install -g @anthropic-ai/claude-code`)
  - Проверить: `claude --version`
- macOS, Linux или Windows (WSL)

---

## Установка (3 шага)

### Шаг 1 — Создать папку для скиллов

Откройте терминал и выполните:

```bash
mkdir -p ~/.claude/skills
```

### Шаг 2 — Скопировать скилл

Перейдите в папку этого пакета (`claude-code-frontend-skill/`), затем:

```bash
cp -r frontend-design/ ~/.claude/skills/
```

**Проверка** — должен появиться файл:

```bash
ls ~/.claude/skills/frontend-design/
# Ожидаемый результат: SKILL.md
```

### Шаг 3 — Скопировать справочники (рекомендуется)

Скопируйте справочные файлы в корень вашей рабочей папки (туда, где будут ваши проекты).

Замените `~/Documents/Projects/` на свой путь:

```bash
cp FRONTEND_DESIGN_SYSTEM.md ~/Documents/Projects/
cp DESIGN_SYSTEM_NEON.md ~/Documents/Projects/
```

> **Зачем?** Claude Code читает файлы рабочей папки при старте и использует их как контекст.
> Это даёт быстрый доступ к справочнику без открытия самого скилла.

## Использование

### Вариант 1 — Явный вызов скилла

В чате Claude Code напишите:

```
/frontend-design
```

### Вариант 2 — Автоматически

Просто опишите задачу — Claude подберёт скилл:

```
Создай лендинг для онлайн-курса по фотографии
Сделай дашборд для аналитики продаж
Создай страницу портфолио для дизайнера
```

### Что произойдёт:

1. Claude задаст **2-3 вопроса** через меню выбора (тип проекта, стиль, секции)
2. Объявит свои решения: шрифт, цвета, mood
3. Сгенерирует **полный проект** — с реальными фото, анимациями, настоящим текстом

## Особенности скилла

### Discovery — вопросы перед кодом

Скилл **всегда** задаёт 2-3 вопроса перед разработкой (через меню), чтобы точно понять задачу.
Это имитирует брифинг с дизайнером — вы выбираете из вариантов, не пишете технические требования.

### Дизайн-решения — самостоятельно

После ответов Claude сам выбирает:
- шрифтовую пару (из 10 курированных)
- цветовую палитру (8 вариантов под разные mood)
- композицию и анимации

Вы не принимаете технических решений — только творческие.

### Работает глобально

После установки скилл доступен в **любом проекте** и **любой папке** — не нужно копировать в каждый проект.

## Если что-то не работает

**Скилл не срабатывает:**
```bash
# Проверить что файл на месте
ls ~/.claude/skills/frontend-design/SKILL.md
```

**Claude Code не установлен:**
```bash
npm install -g @anthropic-ai/claude-code
# или через официальный сайт: https://claude.ai/download
```

**Нет прав на запись:**
```bash
# Создать папку с правами
mkdir -p ~/.claude/skills && chmod 755 ~/.claude/skills
```
