# Project — Личный бренд Карины (AI-интегратор)

## Что это за проект

Персональный веб-сайт Карабут Карины — AI-интегратора для предпринимателей и экспертов.
Проект включает цифровую визитку, два варианта продающего лендинга и вспомогательные материалы личного бренда.
Цель — привлечь клиентов на AI-аудит, внедрение AI-ассистентов и консультации.

## Для кого

**Владелец:** Карабут Карина — AI-интегратор, 15+ лет в бизнесе (фармацевт → администратор → маркетинг → AI).

**Целевая аудитория клиентов:** предприниматели и эксперты 30–55 лет, оборот от 1 млн руб., есть команда и продукт, но нет структуры и роста. Хотят внедрить AI, но не знают как и боятся технического жаргона.

## Стиль дизайна

**Общее направление:** тёмный премиум-минимализм. Строгий, доверительный, технологичный — без лишних украшений.

### Цветовая палитра

| Роль | Значение |
|---|---|
| Фон (основной) | `#080B12` / `#090B0D` / `#111315` |
| Карточки / секции | `#0D1220` / `#14171A` / `#131719` |
| Акцент A (лендинг A) | `#2563EB` — синий |
| Акцент B (лендинг B) | `#10B981` — зелёный / изумрудный |
| Акцент C (визитка) | `#C6A85E` / `#C9A84C` — золото |
| Текст основной | `#F1F5F9` / `#E2E8F0` |
| Текст приглушённый | `#94A3B8` / `#4B5563` |

### Типографика

| Шрифт | Где используется |
|---|---|
| **Inter** | Основной текст, навигация, кнопки — везде |
| **Playfair Display** | Заголовки в визитке — добавляет статус и теплоту |
| Cinzel, Libre Baskerville | Рассматривались в font-preview, не приняты |

### UI-паттерны
- Тонкие бордеры с низкой прозрачностью (`rgba(255,255,255,0.06–0.15)`)
- Плавные анимации `fadeUp` при скролле
- Blur-навигация (`backdrop-filter: blur`)
- Кнопки: solid primary + ghost outline
- Карточки с hover-эффектом свечения
- Мобильная адаптация через flexbox / grid
